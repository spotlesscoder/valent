Description [src]

Returns the full path of a special directory using its logical id.

This function is a wrapper around g_get_user_special_dir() that falls back to the return value of g_get_home_dir() if directory is NULL, and makes a call to g_mkdir_with_parents() to ensure the path exists.

Available since: 1.0

directory

Type: GUserDirectory

A GUserDirectory.

Return value

Type: char*

The path to the specified special directory.

The caller of the function takes ownership of the data, and is responsible for freeing it.
The value is a NUL terminated UTF-8 string.
