Description [src]

A convenience function for creating multiple GMenuItem instances and adding them to the device GMenuModel. Each item is constructed as per one ValentMenuEntry.

Available since: 1.0

Parameters

entries

Type: An array of None

A pointer to the first item in an array of ValentMenuEntry structs.

The length of the array is specified in the n_entries argument.
The data is owned by the caller of the function.
n_entries

Type: int

The length of entries, or -1 if entries is NULL-terminated.
